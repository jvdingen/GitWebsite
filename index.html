Introductie tot Git

Wat is Git?

Git is een versiebeheersysteem dat wordt gebruikt voor het bijhouden van wijzigingen in computerbestanden en het co√∂rdineren van werk tussen meerdere mensen.

Waarom Git gebruiken?

  • Versiebeheer: Houdt de geschiedenis van je project bij.
  • Samenwerking: Maakt het gemakkelijk om samen te werken met anderen.
  • Branching: Mogelijkheid om nieuwe functies uit te proberen zonder de hoofdcode aan te passen.

Basiscommando's

Hier zijn enkele belangrijke Git-commando's:

  • git init - Maak een nieuwe Git-repository aan.
  • git clone [repository-url] - Kopieer een bestaande Git-repository.
  • git add [bestand] - Voeg wijzigingen toe aan de staging area.
  • git commit -m "bericht" - Sla de staged wijzigingen op.
  • git push - Stuur je wijzigingen naar de remote repository.
  • git pull - Haal de laatste wijzigingen van de remote repository op.
  • git status - Beschrijft de huidige status van de git repository. Er wordt verwezen naar de status van bekende en onbekende bestanden.
  • git log - Laat de geschiedenis zien van de git repository
  • git diff - Toont de wijzigingen per file.
